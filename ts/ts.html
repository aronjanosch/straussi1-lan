ETI Launcher

Sync interface

Um das Sync Interface aufzurufen klickt bitte auf folgendes Icon im ETI Launcher

Syncronization hängt (meistens bei 99%)

Einfach rechtklick auf den Packagenamen des Spiels -> pausieren
Danach wieder rechtsklick -> resume

Call of Duty 4 - Modern Warfare

Awaiting Keycode Authorization

Mögliche Fehlerquellen:
  • Mehrere Netzwerkadapter
  • Falsche Netzwerk Reihenfolge -> Lösung
  • +set sv_authorizemode -1 ( set it as a commandline option )

Verbindungsprobleme

Die config_mp.cfg editieren (Call of Duty 4 - Modern Warfare\players\profiles\'profile name'\config_mp.cfg)
seta cl_connectionAttempts "30" hinzufügen/ändern
